This project analyzes global COVID-19 data using Power BI. 
The interactive dashboard visualizes confirmed cases, deaths, 
recoveries, and active cases across countries and regions worldwide.

## Dataset
- Source: Kaggle (COVID-19 Clean Complete Dataset)
- Rows: 49,068 records
- Time Period: January 2020 onwards

## Features
- 4 KPI Cards (Confirmed, Deaths, Recovered, Active)
- Line Chart (Cases over time)
- Donut Chart (Cases by WHO Region)
- Bar Chart (Top countries)
- World Map (Global distribution)

## Key Insights
- Total Confirmed Cases: 829 Million
- Total Deaths: 43 Million
- Total Recovered: 388 Million
- Americas region most affected (48%)
- US, Brazil, Russia top 3 countries

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- CSV Dataset
